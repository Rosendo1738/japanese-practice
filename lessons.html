  • ながら — Doing two things at once

ながら

ながら is used to describe doing two actions at the same time.

How it works

Use the ます-stem of a verb + ながら. 

聞きます → 聞きながら
食べます → 食べながら

Structure

[Verb A (ます-stem)] + ながら + [Verb B]

The second verb is the main action.

Examples

  • 音楽を聞きながら勉強する
    おんがくをききながらべんきょうする
    I study while listening to music.
  • 本を読みながらコーヒーを飲む
    ほんをよみながらこーひーをのむ
    I drink coffee while reading a book.

Important notes

  • Both actions must be done by the same person
  • Actions must happen at the same time
  • Do not use ます-form directly (❌ 聞きますながら)

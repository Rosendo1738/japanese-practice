の Particle

→ Go to の particle exercises

The particle の is used to connect two nouns.

It is commonly used to show:

  • possession
  • description
  • what something belongs to

Basic idea

Think of の as meaning:

“of” / “’s”

Structure

A の B

A describes or owns B.

Common pattern with です

X は A の B です

This lets you make simple statements about people and things.

Examples

  • 私の本です
    わたしの ほんです
    This is my book.
  • これは私のペンです
    これは わたしの ペンです
    This is my pen.
  • 私の猫です
    わたしの ねこです
    This is my cat.
  • 友だちの本です
    ともだちの ほんです
    This is my friend’s book.

Using の to describe people

の is often used to describe what kind of person someone is.

  • 私は学生です
    わたしは がくせいです
    I am a student.
  • 私は大学の学生です
    わたしは だいがくの がくせいです
    I am a university student.
  • 私は大学院の学生です
    わたしは だいがくいんの がくせいです
    I am a graduate student.

More practical examples

  • これは日本語の本です
    これは にほんごの ほんです
    This is a Japanese book.
  • これは学校の先生です
    これは がっこうの せんせいです
    This is a school teacher.
  • あの人は友だちの先生です
    あのひとは ともだちの せんせいです
    That person is my friend’s teacher.

Summary

  • の connects two nouns
  • It can show possession or description
  • It works naturally with X は Y です
